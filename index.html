MIT App Inventor Extensions

This site is a simple directory of some example MIT App Inventor Extensions that you can experiment in your own projects. Note: You *must* use the version of MIT App Inventor located at:

http://extension-test.appinventor.mit.edu

VectorArithmetic created by Ethan Hon
Takes in two vectors and can add them to return a result vector.		Download
ImageProcessor created by Justus Raju
ImageProcessor can do a weighted combine of two images, return the greyscale of an image and do a color blend on an image. 		Download
SoundAnalysis created by Mouhamadou Oumar Sall
Multimedia component that analyzes the pitch of a sound through the microphone and returns it. It can be used as an input to different situations, for instance to control some components with a specific sound whistle(pitch > 500Hz) or clap(pitch > 2000Hz). 		Download
